# Blueme - 나의 학습 도우미

Blueme는 여러분의 학습을 도와주는 프로그램이에요! 

## 주요 기능
1. 프롬프트 저장하기
2. 커스텀 단축키 설정하기
3. 다른 프로그램에서 단축키로 프롬프트 불러오기

## 설치 방법
1. Python이 설치되어 있어야 해요
2. 아래 명령어로 필요한 도구들을 설치해주세요:
```
pip install -r requirements.txt
```

## 사용 방법
1. 프로그램을 실행하면 깔끔한 창이 나타나요
2. 프롬프트를 입력하고 원하는 단축키를 설정해주세요 (예: ctrl+shift+p)
3. "저장하기" 버튼을 누르면 저장돼요
4. 이제 다른 프로그램에서 설정한 단축키를 누르면 자동으로 프롬프트가 입력돼요!

## 단축키 설정 방법
1. 프롬프트 입력창에 원하는 텍스트를 입력해요
2. 단축키 입력창에 원하는 단축키를 입력해요 (예: ctrl+shift+p)
3. 저장하기 버튼을 누르면 끝!

## 저장된 프롬프트 관리
- 저장된 프롬프트는 목록에서 확인할 수 있어요
- 삭제하고 싶은 프롬프트는 선택 후 "선택 삭제" 버튼을 누르면 돼요
